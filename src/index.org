* Steps to run an experiment
  *STEP 1 :* Clone the experiment repository
  #+BEGIN_EXAMPLE
  git clone https://github.com/vitual-labs/<EXPERIMENT>.git
  #+END_EXAMPLE

  *STEP 2 :* Change directory to experiment repository 
  #+BEGIN_EXAMPLE
  cd <EXPERIMENT>
  #+END_EXAMPLE
 
  *STEP 3 :* Get the quiz library 
  #+BEGIN_EXAMPLE
  make -k get-quiz-lib
  #+END_EXAMPLE

  *STEP 4 :* Build the sources
  #+BEGIN_EXAMPLE
  make -k build
  #+END_EXAMPLE
  
  *STEP 5 :* Run the experiment server
  #+BEGIN_EXAMPLE
  make -k run
  #+END_EXAMPLE

  To remove the build 
  #+BEGIN_EXAMPLE
  make -k clean
  #+END_EXAMPLE
